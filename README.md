# Online Job Portal

## Description
Online Job Portal is a web-based application where job seekers can search and apply for jobs, and employers can post and manage job vacancies.

## Modules
- Admin
- Employer
- Job Seeker

## Features
- User Registration and Login
- Job Posting
- Job Search
- Job Application

## Technologies Used
- Java
- MySQL
- HTML
- CSS
- JSP & Servlets

Project is currently under development..
